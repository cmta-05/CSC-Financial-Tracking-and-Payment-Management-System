# ğŸ“¥ 3. How to Upload Files (Simple Walkthrough)

1. Save the file on your laptop with a clear filename.
2. Remove personal notes or temporary comments inside the document.
3. Make sure the file is one of the accepted formats:

   * `.docx`, `.xlsx`, `.pptx`, `.pdf`, `.zip`

### **B. Upload the file to GitHub**

1. Open the repository.
2. Locate the correct folder (ex: `Phase_1_Requirements_and_Specification/Step_2_Organization`).
3. Click **Add file â†’ Upload files**.
